## ğŸ› **Known Issues & Solutions**

<details>
<summary><b>Images not loading?</b></summary>

**Problem:** Images show as broken
**Solution:** Check image paths in `index.html`
```html

<img src="images/Paneer Tikka.png" alt="Paneer Tikka">
```
</details>

<details>
<summary><b>Animations not working?</b></summary>

**Problem:** Elements don't animate
**Solution:** Ensure JavaScript is enabled in browser
</details>

<details>
<summary><b>Mobile layout broken?</b></summary>

**Problem:** Not responsive on mobile
**Solution:** Check viewport meta tag in `<head>`
```html
<meta name="viewport" content="width=device-width, initial-scale=1.0">
```
</details>
